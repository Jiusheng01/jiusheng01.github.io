Project 01

多 Agent 智能旅行助手（trip）

基于 LangGraph 构建的多智能体智能旅行助手，集成 高德地图 MCP 服务，可根据目的地、天数、出行偏好等信息自动生成个性化多日行程。 系统支持景点 POI 搜索、天气查询、步行/驾车/公交路线规划，并整合住宿、交通、餐饮与游览建议， 输出结构化、可执行的旅行计划。这是我在 Agent、工具调用与工作流编排方向的一次完整实践。
